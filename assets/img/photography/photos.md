# Photography manifest

Edit this file to manage the gallery. **One row = one photo.** Row order = display order.

To add a photo: drop the full-size original `<file>.jpg` into this folder, then add a
row below. Push to GitHub — web-sized copies are generated automatically. No HTML to edit.

- **file** — the image's base filename, no extension (e.g. `DSCF1234`).
- **place** — where it was taken (proper nouns; shown as-is in every language).
- **year** — when it was taken.
- **caption** — one sentence describing the photo. Used as the lightbox caption and as
  the image's alt text for accessibility. Keep it factual and language-neutral.
